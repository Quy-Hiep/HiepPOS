Hướng Dẫn Cài Đặt Máy In

Cài Đặt Trên Windows

  1. Tải Driver

    Chọn driver phù hợp với hệ điều hành của bạn từ danh sách ở trên

  2. Giải Nén File

    Sau khi tải xong, giải nén file .zip vào một thư mục trên máy tính

  3. Chạy Cài Đặt

    Tìm và chạy file Setup.exe hoặc Install.exe trong thư mục giải nén

  4. Kết Nối Máy In

    Kết nối máy in qua USB hoặc kết nối mạng theo hướng dẫn trên màn hình

  5. Hoàn Tất

    Khởi động lại máy tính nếu được yêu cầu. Driver đã sẵn sàng để sử dụng

Cài Đặt Trên macOS

  1. Tải Driver

    Chọn driver macOS từ danh sách ở trên

  2. Mở File DMG

    Nhấp đúp vào file .dmg để mở. Bạn sẽ thấy biểu tượng cài đặt

  3. Chạy Installer

    Nhấp đúp vào file installer và nhập mật khẩu admin nếu yêu cầu

  4. Kết Nối Máy In

    Kết nối máy in USB hoặc qua mạng. macOS sẽ tự động phát hiện

  5. Xác Nhận Cài Đặt

    Vào System Preferences > Printers & Scanners để xác nhận máy in đã sẵn sàng
